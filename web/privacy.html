Privacy Policy

Last updated: April 13, 2026

Quizza offers optional account registration via Supabase. If you create an account, we store your email address, username, and hashed password. You may also play as a guest without creating an account.

Display names are used within game sessions. Guest names are not stored after the session ends. Registered account data is retained until you request deletion.

Quiz questions are generated using the DeepSeek API. When a game starts, the host's study material is sent to DeepSeek's servers to generate questions. No personal data (names, emails) is included in these requests.

If you upload a file (TXT, MD, CSV, JSON, PDF, DOC, DOCX, XLSX, PPTX, HTML, or XML) as an AI question source, the file is processed in memory to extract text and is deleted from the server immediately after processing. File content is not retained.

We use Google Analytics and Google AdSense on this site, which may set cookies and collect anonymous usage data in accordance with Google's privacy policies.

We use localStorage in your browser to save preferences such as sound mute state. No tracking cookies are set by Quizza itself.

By using Quizza you acknowledge this privacy policy. If you have questions, contact us at vakaris.tech@gmail.com.
